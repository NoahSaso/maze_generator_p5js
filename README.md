maze_generator_p5js
===================

Idea from [https://www.youtube.com/watch?v=HyK_Q5rrcr4](https://www.youtube.com/watch?v=HyK_Q5rrcr4), original code but using same algorithm. Shoutout to Daniel Shiffman for being awesome.
